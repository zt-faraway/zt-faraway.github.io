Desriction

一直想搭建自己的blog,可是一直耽误拖延,直到今天终于开始了blog之旅,看了各种blog搭建工具和方案 最终选择了Hexo(静态页面发布框架) + GithubPages,搭建过程由于对前端相关技术不是很懂,进行过各种尝试,中间吃了不少苦头,现记录相关操作,以做备忘.本人使用系统为macOs,故以下操作均是基于Mac的.

Tools Prepare

  • Home Brew安装
  • Nvm安装
  • Node安装

Home Brew 安装

HomeBrew,系统一般已经自带(MacOs的安装包管理工具,可进行相关软件的安装、更新、卸载),若不确定,是否已经安装,请使用brew -v 若有相关版本信息输出,则说明已经安装.反之,请复制如下指令,在终端执行(偶尔可能应为被墙的原因安装失败,如果失败可进行多次尝试,或者安装翻墙软件Latain)

Nvm 安装

总的来说,nvm时node的版本管理工具,有点类似于 Python 的 virtualenv 或者 Ruby 的 rvm,每个node版本的模块都会被安装在各自版本的沙箱里面(因此切换版本后模块需重新安装),由于想统一管理node,故选择nvm.
